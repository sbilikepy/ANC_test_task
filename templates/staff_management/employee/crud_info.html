Management FAQ

This section provides information on how to manage employee records. Only authenticated users can create, read, update, and delete employee records.

  • Create an Employee

    To create a new employee, navigate to the this page.

  • Read Employee Information

    Authenticated users can view employee details by accessing the links in the hierarchy tree or the employee list table.

  • Update Employee Information

    To update an employee's information, go to the employee's profile page.

  • Delete an Employee

    To delete an employee, visit the employee's profile page and use the delete option.
